
# Sistema Bancário  em Python

Projeto bancário simples desenvolvido em Python utilizando terminal.


## Funcionalidades

- Acessar o sistema
- Criar uma conta
- Fazer Depositos
- Realizar Saques
- Realizar Transferencias 
- Ver Extrato
## Tecnologias utilizadas

- Python


## Objetivo

Projeto criado para praticar:

- Lógica de programação
- Listas e dicionários
- Funções
- CRUD básico
- Organização de código
